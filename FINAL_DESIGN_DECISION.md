## 助记词本地化 - 最终确认

### 核心逻辑（最终版）

**不是翻译 BIP39，而是用该语言的地名直接替代 BIP39 词**

#### 流程图

```
用户选择语言
  ↓
┌─────────────────────────────────────┐
│                                     │
├─ 英文选择                          │
│  └─ 使用 BIP39 标准英文词          │
│     abandon, ability, able...       │
│                                     │
├─ 非英文选择（中文、日文等）       │
│  ├─ 中文 → 2048 个中文地名          │
│  │        北京、上海、广州、深圳...  │
│  │                                   │

│  │        (日语发音/汉字)           │
```

#### 【中文简体 (zh-cn)】

**词库内容：2048 个中国主要城市和地名**

```json

  "宁波",
  "东莞",
  "无锡",
  "福州",
  ...（共 2048 个中国地名）
]
```

**词库选择标准：**
- ✅ 中国省会城市（全国 34 个）
- ✅ 地级市（全国 294 个）
- ✅ 县级市和知名县城
- ✅ 覆盖全球（如 Beijing, Shanghai, Guangzhou... 的中文名）

**为什么选择地名而不是抽象词汇？**
1. **人类记忆** — 地名比"金木水火"更容易联想和记忆
2. **文化认同** — 用户对本地地名有深刻认识
3. **唯一性** — 每个地名都是独特的，不易混淆
4. **国际通用** — 地名在全球都能被认识

## 关键决策理由

### 为什么用地名而不是抽象概念？

**之前考虑的方案：** 五行八卦（金、木、水、火、土）

**放弃原因：**
- ❌ 只有 5 个基本词，需要大量组合才能达到 2048
- ❌ 用户容易混淆和遗忘
- ❌ 不符合国际化要求

**最终方案：** 2048 个地名

**优点：**
- ✅ 每个词都是独特的、有具体含义的
- ✅ 用户对地名有现实感受（可以想象这个城市）
- ✅ 国际通用（地名在所有文化中都通用）
- ✅ 易于多语言扩展
- ✅ 防止重复和混淆
