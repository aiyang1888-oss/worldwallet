## 修复后的测试流程

### 【测试 #1】创建钱包 - 中文

**前置条件：** Bug #2 已修复，中文词库正确显示

```
1. 访问 http://localhost:8766/wallet.html
2. 点击"🚀 创建新钱包"
3. 选择"✨ 创建新钱包"
4. 选择语言：中文
5. 选择词数：12 词
6. 查看显示的助记词
   ✅ 预期显示：金、木、水、火、土、天、地、日、月、星、...
   ❌ 不应该显示：地名拼凑（北朝府、库纳尔...）
7. 点击"✅ 我已抄写，开始验证"
8. 输入随机抽到的 3 个词
9. 验证成功 → 钱包创建完成
10. 设置 PIN 码（6 位数字）
```

**验收标准：**
- ✅ 显示中文词汇（五行八卦 + 常用字）
- ✅ 12 个不同的词
- ✅ 验证通过
- ✅ PIN 设置成功

## 风险和限制

### 已知限制

1. **服务稳定性** — 8766 端口的 Python HTTP 服务有时会自动停止
   - 缓解：每次测试前检查服务状态

2. **Cursor 初始化缓慢** — acpx cursor exec 有时初始化时间长
   - 缓解：预留充足的等待时间

3. **浏览器缓存** — 页面修改后需清除缓存
   - 缓解：使用 ?nocache= 参数或 Ctrl+Shift+R

### 待解决的问题

1. 万语地址是否正确显示（待测试）
2. 多链 RPC 连接是否正常（需真实数据）
3. 硬钱包支持（Phase 2）
