第一章 软件概述

1.1 软件简介

WorldToken 多语言数字钱包系统（以下简称"本软件"）是一款基于区块链技术的去中心化数字资产管理工具，其核心创新在于采用汉字作为助记词，实现了数字钱包密钥的中文本地化表达。

本软件支持以太坊（ETH）、波场（TRX/USDT-TRC20）、比特币（BTC）三条主流公链，提供创建钱包、导入钱包、查询余额、资产转账等完整功能。软件以单一HTML文件形式交付，无需安装，在本地浏览器中运行，私钥从不上传服务器，保障用户资产安全。

1.2 主要特点

  • 汉字助记词：全球首创以汉字城市名为助记词的钱包标准，用户可用母语记忆私钥
  • 多链支持：一套助记词同时派生 ETH、TRX、BTC 三链地址
  • 11种语言：支持中文、英文、日文、韩文、西班牙文、法文、阿拉伯文、俄文、葡萄牙文、印地文等词库切换
  • 完全本地化：无服务器依赖，所有操作在用户设备本地完成
  • 零安装：单HTML文件，支持手机、电脑浏览器直接运行

1.3 技术架构

模块技术实现
助记词生成SHA-256哈希 + 汉字词库映射，符合BIP39安全标准
密钥派生BIP32/BIP44分层确定性钱包协议
以太坊支持ethers.js v5.7 库

余额查询TronGrid API / ETH公共节点 / mempool.space
前端界面HTML5 + CSS3 + Vanilla JavaScript

第三章 安装与启动

3.1 获取软件

本软件通过以下方式获取：

  1. 访问官方网站：www.worldtoken.cc
  2. 点击导航栏中的"进入钱包"按钮
  3. 也可直接访问：www.worldtoken.cc/wallet.html

3.2 本地部署（可选）

用户可将 wallet.html 文件下载到本地，无需网络即可使用钱包基础功能（余额查询、转账需要网络）。

  1. 在浏览器中按 Ctrl+S（Mac：Command+S）保存完整网页
  2. 保存为 wallet.html 单文件
  3. 双击打开即可在本地浏览器运行

3.3 首次启动

打开软件后，界面将显示欢迎页面，提供以下选项：

  • 🚀 创建新钱包：生成全新密钥对
  • 📥 导入已有钱包：通过助记词恢复已有钱包
注意：首次使用请务必仔细阅读安全提示，妥善保管助记词，任何人获得助记词即可控制您的资产。

第四章 创建钱包

4.1 创建流程

  1. 点击首页"🚀 创建新钱包"按钮
  2. 在弹出的选项中选择"✨ 创建新钱包"
  3. 系统自动生成12个汉字助记词（可选12/15/18/21/24词）
  4. 仔细抄写助记词，建议手写在纸上，一字不差
  5. 点击"✅ 我已抄写，开始验证"进行验证
  6. 按提示顺序输入指定位置的词，通过验证后钱包创建完成

4.2 助记词选择

软件提供5种助记词长度，安全性随长度增加而提升：

长度熵值安全强度适用场景
12词128位2¹²⁸日常使用，小额资产
15词160位2¹⁶⁰一般使用
18词192位2¹⁹²较高安全需求
21词224位2²²⁴高价值资产
24词256位2²⁵⁶最高安全级别

4.3 派生地址说明

创建钱包后，系统将自动生成以下三条链的地址：

  • ETH 地址：以"0x"开头，42位十六进制字符串，用于接收 ETH 及 ERC-20 代币
  • TRX 地址：以"T"开头，34位 Base58 字符串，用于接收 TRX 及 USDT-TRC20
  • BTC 地址：以"1"或"bc1"开头，用于接收比特币

第五章 导入已有钱包

5.1 通过助记词导入

  1. 点击首页"📥 导入已有钱包"
  2. 在文本框中输入助记词，词与词之间用空格分隔
  3. 支持12/15/18/21/24个词的助记词
  4. 支持标准BIP39英文助记词（兼容MetaMask、imToken等主流钱包）
  5. 支持本软件生成的汉字助记词
  6. 点击"✅ 导入钱包"完成导入

5.2 兼容性说明

钱包类型是否兼容说明
MetaMask 助记词✅ 完全兼容标准BIP39英文助记词
imToken 助记词✅ 完全兼容标准BIP39英文助记词

第六章 资产管理

6.1 查看资产

创建或导入钱包后，进入主界面可查看：

  • USDT-TRC20 余额（波场链）
  • TRX 余额（波场链）
  • ETH 余额（以太坊链）
  • BTC 余额（比特币链）
  • 各资产当前法币价值（美元计价）
  • 资产总价值汇总

6.2 收款地址

  1. 点击底部导航"地址"标签
  2. 选择收款链（TRX/ETH/BTC）
  3. 系统显示对应链的收款地址及二维码
  4. 点击地址可一键复制
  5. 将二维码展示给付款方扫描即可收款

6.3 交易记录

系统通过 TronGrid API 自动获取 TRX/USDT-TRC20 的链上交易历史，显示：

  • 交易时间
  • 交易金额
  • 交易方向（收入/支出）
  • 对方地址
  • 交易哈希（可在区块链浏览器查询）

第七章 转账功能

7.1 转账操作步骤

  1. 进入主界面，点击"📤 转账"按钮
  2. 选择转账币种（USDT-TRC20 / TRX / ETH）
  3. 输入收款地址（或扫描对方二维码）
  4. 输入转账金额
  5. 确认手续费信息
  6. 点击"确认转账"，系统在本地完成私钥签名
  7. 签名后广播到对应区块链网络
  8. 等待区块链确认（TRX约3秒，ETH约15秒，BTC约10分钟）

7.2 手续费说明

币种手续费类型收取方
USDT-TRC20消耗 TRX 带宽/能量波场网络
TRX消耗 TRX 带宽波场网络
ETHGas 费（ETH计价）以太坊网络
重要提示：转账前请仔细核对收款地址，区块链转账一旦广播无法撤回。建议首次向新地址转账时先发送小额测试。

第九章 礼物口令功能

9.1 功能简介

礼物口令功能允许用户生成一个加密口令，通过任何社交媒体分享给指定收款人，对方输入口令后即可领取相应数字资产。

9.2 发送礼物

  1. 点击底部导航"🎁 礼物"标签
  2. 选择礼物类型（固定金额/随机）
  3. 输入礼物金额
  4. 设置有效期（1-72小时）
  5. 点击"🎁 生成礼物口令"
  6. 复制口令分享给收款人

9.3 领取礼物

  1. 收到口令后，打开 WorldToken 钱包
  2. 点击"🎊 领取礼物"按钮
  3. 输入口令
  4. 点击"立即领取"完成领取

第十章 安全注意事项

10.1 助记词安全

⚠️ 极其重要：助记词是您数字资产的唯一凭证，等同于银行密码+账号的总和。
  • ✅ 手写在纸上，保存于安全位置
  • ✅ 制作多份备份，存放于不同地点
  • ✅ 考虑使用防水防火的金属助记词板
  • ❌ 切勿截图保存（手机被盗或入侵即损失资产）
  • ❌ 切勿发送到任何聊天软件或邮件
  • ❌ 切勿输入到任何不信任的网站或软件
  • ❌ 切勿告知任何人，包括官方客服

10.2 使用环境安全

  • 避免在公共WiFi网络下进行转账操作
  • 确保设备无恶意软件和键盘记录程序
  • 使用完毕后及时关闭钱包页面
  • 不在他人设备上登录自己的钱包

10.3 防诈骗提示

  • 官方永远不会索要您的助记词或私钥
  • 任何要求您输入助记词的网站均为钓鱼网站

第十一章 常见问题解答

Q1：我的资产存在哪里？软件会保管我的资产吗？

您的资产存储在区块链上，本软件不保管任何资产。私钥（助记词）由您自己保管，本软件仅提供界面操作工具。

Q2：软件停止运营会影响我的资产吗？

不会。区块链是去中心化网络，即使本软件停止运营，您持有助记词，即可通过任何兼容BIP39标准的钱包（如MetaMask）恢复资产。

Q3：忘记助记词怎么办？

助记词无法找回。这是区块链去中心化的核心特性，没有任何中心机构能帮您重置。因此备份助记词极为重要。

Q4：为什么余额查询有时失败？

余额查询依赖第三方API接口，偶发的网络波动或接口限流可能导致查询失败。刷新后重试或稍后再查即可。

Q5：汉字助记词与英文助记词的钱包地址是否相同？

不相同。不同词库生成的助记词对应不同的私钥，因此地址也不同。请勿混淆使用。

Q6：手机能使用吗？

可以。本软件已针对移动设备进行优化，支持 iOS Safari 和 Android Chrome 浏览器正常使用。
