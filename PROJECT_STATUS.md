## 技术架构（确认）

### 助记词本地化流程

```
用户选择语言
  ↓
┌─────────────────────────────────┐
│ 英文选择 → BIP39 标准词          │
│ 中文选择 → 中文词库（2048 地名）  │
│ 日文选择 → 日文词库              │
│ 其他语言 → 该语言词库            │
└─────────────────────────────────┘
  ↓
系统内部生成 BIP39 英文词（ethers.js）
  ↓
查表转换为本地语言地名
  ↓
显示给用户（用户看到母语）
  ↓
用户验证输入本地语言
  ↓
反向查表转换为 BIP39
  ↓
派生私钥（完全标准 BIP39）
```

### 万语地址系统

**格式：** `[8数字]-[10个本地语言地名]-[8数字]`

**示例：**
```
中文：12345678-北京上海广州深圳成都重庆杭州武汉西安南京-87654321
日文：12345678-北京上海広州深圳成都重慶杭州武漢西安南京-87654321
英文：12345678-abandon-ability-able-about-above-absent-absorb-abstract-absurd-abuse-87654321
```
